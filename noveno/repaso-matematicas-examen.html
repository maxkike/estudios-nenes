Suma por Diferencia

Tema 1
Producto de suma por diferencia
(a + b)(a − b) = a2 − b2
Regla: El producto de la suma de dos cantidades por su diferencia es igual al cuadrado del minuendo menos el cuadrado del sustraendo (cuadrado del primero menos cuadrado del segundo).
Ejemplos de clase
(3x + 2y)(3x − 2y)
a = 3x, b = 2y
= (3x)2 − (2y)2 = 9x2 − 4y2
(m + 7)(m − 7)
a = m, b = 7
= m2 − 49
(5a2b + 3c)(5a2b − 3c)
a = 5a2b, b = 3c
= 25a4b2 − 9c2
(23x + 14y)(23x − 14y)
a = 23x, b = 14y
= 49x2 − 116y2

Producto de binomios con término común
(x + a)(x + b) = x2 + (a+b)x + ab
Regla: El cuadrado del término común, más el producto del término común por la suma de los términos no comunes, más el producto de los términos no comunes.
Ejemplos de clase
(x + 3)(x + 5)
Término común: x; No comunes: 3 y 5
= x2 + (3+5)x + (3)(5) = x2 + 8x + 15
(m − 2)(m − 7)
Término común: m; No comunes: −2 y −7
= m2 + (−2−7)m + (−2)(−7) = m2 − 9m + 14
(2x + 3)(2x + 8)
Término común: 2x; No comunes: 3 y 8
= (2x)2 + (3+8)(2x) + (3)(8) = 4x2 + 22x + 24
(a2 − 4)(a2 + 6)
Término común: a²; No comunes: −4 y 6
= (a2)2 + (−4+6)a2 + (−4)(6) = a4 + 2a2 − 24

Cubo de un binomio
(a + b)3 = a3 + 3a2b + 3ab2 + b3
(a − b)3 = a3 − 3a2b + 3ab2 − b3
Regla: El cubo del 1°, más 3 veces el cuadrado del 1° por el 2°, más 3 veces el 1° por el cuadrado del 2°, más/menos el cubo del 2°.

Patrón de signos: Todos positivos si es suma; alternados comenzando con positivo si es resta.
Ejemplos resueltos
(a + b)3
Suma: todos positivos
= a3 + 3a2b + 3ab2 + b3
(2x + 1)3
a=2x, b=1. Suma.
= (2x)3 + 3(2x)2(1) + 3(2x)(1)2 + 13
= 8x3 + 12x2 + 6x + 1
(m − 2)3
a=m, b=2. Resta: +−+−
= m3 − 3m2(2) + 3m(2)2 − 23
= m3 − 6m2 + 12m − 8
(3a + 4b)3
a=3a, b=4b. Suma.
= (3a)3 + 3(3a)2(4b) + 3(3a)(4b)2 + (4b)3
= 27a3 + 108a2b + 144ab2 + 64b3

¿Qué son los cocientes notables?

Son divisiones de polinomios que se pueden simplificar sin realizar la división larga. Se obtienen cuando dividimos expresiones algebraicas especiales.

Casos principales:
1. Diferencia de cuadrados entre diferencia de bases
2. Diferencia de potencias pares entre diferencia o suma de bases
3. Suma o diferencia de potencias impares entre suma o diferencia de bases
4. Suma de potencias pares cuando la potencia es impar
Ejemplos básicos
x2 − 4x − 2
x²−4 = (x+2)(x−2), entonces:
= x + 2
a3 − b3a − b
Diferencia de cubos entre diferencia
= a2 + ab + b2
m4 − 16m − 2
m⁴−16 = (m²)²−4² = (m²+4)(m²−4) = (m²+4)(m+2)(m−2)
= (m2+4)(m+2)
p5 + q5p + q
Suma de 5ª potencia entre suma (5 es impar)
= p4 − p3q + p2q2 − pq3 + q4

Diferencia de potencias pares
a2n − b2nan − bn = an + bn
Cuando n es PAR:
• Diferencia entre diferencia → suma
• Diferencia entre suma → se descompone más
Ejemplo: (a⁴−b⁴)÷(a−b) = a³ + a²b + ab² + b³
También: (a⁴−b⁴)÷(a+b) = a³ − a²b + ab² − b³

Suma o diferencia de potencias impares (Parte 2)

¿Qué es?

El factor común monomio es el factor que se repite en todos los términos (el MCD de todos ellos). Se extrae fuera del paréntesis.

ax + ay + az = a(x + y + z)
Cómo hacerlo:
1. Busca el MCD numérico de los coeficientes.
2. Busca la letra común con el menor exponente.
3. Divide cada término entre ese MCD. El resultado va dentro del paréntesis.
Ejemplos de clase
4x + 4y − 4z − 4w + 4m
MCD = 4
= 4(x + y − z − w + m)
9x2y3z3 − 27xy4 − 18x3y2z2
MCD = 9xy2
= 9xy2(xyz3 − 3y2 − 2x2z2)
4a5b − 8a4b2 − 2a3b3 + 6a2b − 12ab2
MCD = 2ab
= 2ab(2a4 − 4a3b − a2b2 + 3a − 6b)
72a3x3 − 5a2x4 + 14ax5
MCD = 14ax3
= 14ax3(14a2 − 20ax + x2)

¿Qué es?

El factor que se repite en todos los términos es un binomio o polinomio completo.

a(m+n) + b(m+n) + c(m+n) = (m+n)(a+b+c)

¿Qué es?

Cuando no hay factor común a todos los términos, se forman grupos donde sí existe factor común, y luego se extrae el factor común polinomio.

ax + bx + ay + by
= (ax + bx) + (ay + by)
= x(a+b) + y(a+b)
= (a+b)(x+y)
⚠️ Cuando un grupo va precedido de signo menos, el signo de todos sus términos cambia al abrir el paréntesis.

Variación del factor común polinomio donde los grupos producen un binomio idéntico en todos los términos.

A(B+C) − D(B+C) = (B+C)(A−D)
💡 Si un binomio aparece con signos opuestos: −(a−b) = (b−a). Se pueden igualar cambiando el signo exterior.

Trinomio Cuadrado Perfecto (TCP)
a2 + 2ab + b2 = (a+b)2
a2 − 2ab + b2 = (a−b)2
3 condiciones para ser TCP:
1. El 1° y 3° término son cuadrados perfectos positivos.
2. El término del medio = 2 × √(1°) × √(3°)
3. El signo del 1° y 3° término es positivo.

Para factorizar: extraes la raíz del 1° y del 3°, y el signo del 2° término te dice si el binomio va con + o −.
Ejemplos resueltos
x2 + 6x + 9
√(x2)=x, √9=3, 2·x·3=6x ✓
= (x+3)2
4a2 − 12ab + 9b2
√(4a2)=2a, √(9b2)=3b, 2·2a·3b=12ab ✓
= (2a−3b)2
16m4 + 40m2n3 + 25n6
a=4m2, b=5n3
= (4m2+5n3)2
19x4 − 23x2y + y2
a=13x2, b=y
= (13x2 − y)2
⚠️ TCP + Diferencia de cuadrados: Si obtienes (algo)2 − (algo)2, aplica la diferencia de cuadrados encima del TCP. Ej: 4c2 − (a−b)2 = (2c+a−b)(2c−a+b)

Trinomio de la forma x² ± bx ± c
x2 + bx + c = (x + p)(x + q)
Busca dos números p y q tales que:
• p + q = b (suma = coeficiente del término medio)
• p · q = c (producto = término independiente)
Tabla de signos

Trinomio ax² + bx + c — Método del Aspa
ax2 + bx + c = (px + r)(qx + s)
Se buscan p, q, r, s tales que:
• p · q = a (primer coeficiente)
• r · s = c (término independiente)
• p·s + r·q = b (suma de los productos en aspa = término medio)
Cómo usar el Aspa — paso a paso
Ejemplo: 6x2 + 7x + 2
Prueba (2x+1)(3x+2):
Aspa: 2×2 + 1×3 = 4+3 = 7 ✓
= (2x+1)(3x+2)
Ejemplo: 6x2 − 5x − 4
Prueba (2x+1)(3x−4):
Aspa: −4×2 + 1×3 = −8+3 = −5 ✓
= (2x+1)(3x−4)
Ejemplo: 2x2 + 5x + 3
Prueba (2x+3)(x+1):
Aspa: 3×1 + 1×2 = 3+2 = 5 ✓
= (2x+3)(x+1)
Ejemplo: 3a2 − 7a + 4
Prueba (3a−4)(a−1):
Aspa: −4×1 + (−1)×3 = −4−3 = −7 ✓
= (3a−4)(a−1)

Trinomio Incompleto — Reducción y Sustracción

Un trinomio incompleto le falta un término. Se convierte en completo sumando y restando el mismo valor, luego se aplica TCP + diferencia de cuadrados.

a4 + a2b2 + b4 = (a2+ab+b2)(a2−ab+b2)
Ejemplos paso a paso
x4 + x2 + 1
Falta término: suma y resta x2
= x4 + 2x2 + 1 − x2
= (x2+1)2 − x2
= (x2+1+x)(x2+1−x)
= (x2+x+1)(x2−x+1)
4a4 + 1
Suma y resta 4a2
= 4a4 + 4a2 + 1 − 4a2
= (2a2+1)2 − (2a)2
= (2a2+2a+1)(2a2−2a+1)

Combinación de Técnicas de Factorización

Muchos ejercicios requieren aplicar varias técnicas consecutivas. La clave es identificar cuál aplica en cada paso y seguir hasta que todos los factores sean primos entre sí.

Orden sugerido:
1. ¿Hay factor común monomio? → Sácalo primero.
2. ¿Lo que queda es dif. cuadrados / dif. cubos / TCP / trinomio?
3. Continúa factorizando cada factor hasta que no pueda más.
Ejemplo 1 — TCP + Diferencia de cuadrados
4c2 − a2 + 2ab − b2
Observamos TCP en a2 − 2ab + b2:
= 4c2 − (a2 − 2ab + b2)
= 4c2 − (a−b)2
= [2c+(a−b)][2c−(a−b)]
= (2c+a−b)(2c−a+b)
Ejemplo 2 — Agrupación + Diferencia de cubos
x4 − 8x + x3 − 8
Agrupamos: (x4 − 8x) + (x3 − 8)
FCM: x(x3−8) + (x3−8)
FCP: (x3−8)(x+1)
Dif. cubos: (x−2)(x2+2x+4)(x+1)

Factorización por Evaluación — Teorema del Factor
Si P(a) = 0 → (x − a) es factor de P(x)
Pasos:
1. Prueba con los divisores del término independiente (±1, ±2, ±3…)
2. Cuando P(a)=0 encontraste un factor: (x − a)
3. Divide el polinomio entre (x − a) usando división sintética (Ruffini)
4. El cociente puede seguirse factorizando.
Ejemplo resuelto
P(x) = x3 − 2x2 − 5x + 6
Prueba x=1: 1 − 2 − 5 + 6 = 0 ✓
Divide entre (x−1) → cociente: x2 − x − 6
Factoriza cociente (forma x2+bx+c): (x−3)(x+2)
= (x−1)(x−3)(x+2)

Ecuaciones lineales con una incógnita

Una ecuación lineal tiene la variable con exponente 1. Se resuelve aislando la variable aplicando propiedades de igualdad.

ax + b = c → x = c − ba
Propiedades:
• Suma: sumar o restar lo mismo en ambos lados no cambia la ecuación.
• Producto: multiplicar o dividir por el mismo número (≠0) no la cambia.
• Lo que pasa de un lado al otro cambia de signo (suma ↔ resta) o de operación (× ↔ ÷).
Ejemplos resueltos
3(2x−5) − 4x + 3 = x − 5(x−3)
6x − 15 − 4x + 3 = x − 5x + 15
2x − 12 = −4x + 15
6x = 27
x = 9/2
5(x+2) − 3(x−1) = 2x + 14
5x+10−3x+3 = 2x+14
2x+13 = 2x+14
0 = 1 → Sin solución

Ecuaciones con fracciones algebraicas

Se elimina el denominador multiplicando todos los términos por el MCM de los denominadores.

x2 + x3 = 5 → ×6 → 3x + 2x = 30 → x = 6
Pasos:
1. Calcula el MCM de todos los denominadores.
2. Multiplica cada término por el MCM.
3. Simplifica y resuelve sin fracciones.
4. Verifica: el valor de x no debe anular ningún denominador.
Ejemplos
2x3 − x+14 = x6 + 1
MCM = 12
8x − 3(x+1) = 2x + 12
5x − 3 = 2x + 12
x = 5
x+1x−1 − x−1x+1 = 4x2−1
MCM = (x−1)(x+1)
(x+1)2 − (x−1)2 = 4
4x = 4 → x=1
⚠️ x=1 anula denominador → Sin solución

Conjuntos — Conceptos fundamentales

Un conjunto es una colección bien definida de objetos llamados elementos.

Por extensión:
A = {1, 2, 3, 4, 5}
Se listan todos los elementos.
Por comprensión:
A = {x / x es número natural < 6}
Se describe la propiedad.
Tipos de conjuntos
Vacío (∅): Sin elementos.
Unitario: Un solo elemento.
Finito: Se pueden contar sus elementos.
Infinito: No se pueden contar. (ℕ, ℤ, ℝ)
Universal (U): Contiene todos los del contexto.
Subconjunto (⊆): A⊆B si todo elemento de A está en B.
Notación
a ∈ A → "a pertenece a A"
a ∉ A → "a no pertenece a A"
A ⊆ B → "A es subconjunto de B"
n(A) = número de elementos de A

✏️ Practica — marca para ver solución

Relaciones entre conjuntos
Igualdad (A = B):
Mismos elementos exactamente.
{1,2,3} = {3,2,1} ✓
Equivalencia (A ~ B):
Mismo número de elementos.
n(A) = n(B)
Subconjunto (A ⊆ B):
Todo elemento de A está en B.
{1,2} ⊆ {1,2,3,4}
Subconjunto propio (A ⊂ B):
A ⊆ B y además A ≠ B.
B tiene al menos un elemento más.
Número de subconjuntos: Si n(A) = n, entonces A tiene 2n subconjuntos (incluyendo ∅ y el propio A).
💡 El conjunto vacío ∅ es subconjunto de todo conjunto. Todo conjunto es subconjunto de sí mismo.

✏️ Practica — marca para ver solución

Operaciones entre conjuntos
Unión (A ∪ B):
Todos los de A o B (o ambos).
{1,2,3} ∪ {3,4,5} = {1,2,3,4,5}
Intersección (A ∩ B):
Los que están en A y en B.
{1,2,3} ∩ {3,4,5} = {3}
Diferencia (A − B):
Los de A que no están en B.
{1,2,3} − {3,4,5} = {1,2}
Complemento (A'):
Los del Universal que no están en A.
Si U={1..5}, A={1,2} → A'={3,4,5}
Dif. simétrica (A △ B):
Los de A o B, pero no en ambos.
= (A−B) ∪ (B−A)
Producto cartesiano (A × B):
Todos los pares (a,b).
n(A×B) = n(A) × n(B)
Fórmula cardinal: n(A ∪ B) = n(A) + n(B) − n(A ∩ B)

Relaciones

Una relación de A en B es un subconjunto del producto cartesiano A × B. Se forma con pares ordenados (x, y).

Dominio:
Primeros elementos de los pares.
Dom = {x / (x,y) ∈ R}
Rango/Recorrido:
Segundos elementos de los pares.
Ran = {y / (x,y) ∈ R}
Propiedades de equivalencia
Reflexiva:
(a,a) ∈ R para todo a.
Cada elemento se relaciona consigo mismo.
Simétrica:
Si (a,b) ∈ R, entonces (b,a) ∈ R.
Transitiva:
Si (a,b) ∈ R y (b,c) ∈ R, entonces (a,c) ∈ R.
Relación de equivalencia:
Cumple las 3 propiedades a la vez.

✏️ Practica — marca para ver solución

Funciones

Una función es una relación donde a cada elemento del dominio le corresponde exactamente uno del rango.

f : A → B    ó    y = f(x)
Prueba vertical: Si una línea vertical corta la gráfica en más de un punto, NO es función.

Evaluación: Para hallar f(a), sustituyes x por a en la expresión.
Evaluación de funciones
f(x) = 3x2 − 2x + 1
f(2) = 3(4) − 2(2) + 1 = 12 − 4 + 1 = 9
f(−1) = 3(1) − 2(−1) + 1 = 3 + 2 + 1 = 6
f(0) = 0 − 0 + 1 = 1
Es función:
f(x) = 2x + 3
Cada x tiene un único y.
NO es función:
x2 + y2 = 25 (círculo)
Para x=0: y=5 y y=−5 (dos valores)

✏️ Practica — marca para ver solución

Proporcionalidad Directa e Inversa
Directa: y = kx
Si x ↑, y ↑ en la misma proporción.
k = yx (constante)
Gráfica: línea recta por el origen.
Inversa: y = kx
Si x ↑, y ↓ proporcionalmente.
k = x · y (constante)
Gráfica: hipérbola.
Ejemplos resueltos
Directa: Persona viaja a 10 km/h constante.
d = 10t → k = 10
En 5h: d = 10 × 5 = 50 km
Inversa: 5 pintores tardan 10 días.
k = 5 × 10 = 50
2 pintores: t = 502 = 25 días
Del libro (función del congelador):
f(t) = 202t+1 − 2 → temperatura cae al aumentar t → inversa.
Para t=0: f(0) = 20 − 2 = 18°

Inecuaciones de primer grado

Una inecuación es una desigualdad con variable. La solución es un conjunto de valores, no uno solo.

Símbolos:
< menor que
> mayor que
≤ menor o igual
≥ mayor o igual
Propiedades:
• Se suma/resta igual a ambos lados.
• Al multiplicar o dividir por negativo, el signo de desigualdad se invierte.
⚠️ Regla clave: Si divides por un número negativo, el signo cambia de dirección.
Ej: −2x > 6 → x < −3
Ejemplos resueltos
2x − 3 > 7
2x > 10
x > 5
Solución: (5, +∞)
−3x + 5 ≤ 11
−3x ≤ 6
x ≥ −2 (se invierte al ÷ por −3)
Solución: [−2, +∞)
2(x+1) − 3 < 5(x−2) + 7
2x+2−3 < 5x−10+7
2x−1 < 5x−3
−3x < −2
x > 23
−1 ≤ 2x+1 ≤ 5 (doble)
−2 ≤ 2x ≤ 4
−1 ≤ x ≤ 2
Solución: [−1, 2]
📐 En la recta numérica: círculo abierto ○ = valor no incluido (< ó >) | círculo cerrado ● = valor incluido (≤ ó ≥)
